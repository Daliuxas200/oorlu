About oor.lu - free url shortener.

Backend:

This url shortener was created with django-REST (Python). It utilizes combined powers of PostgreSQL and Redis cache for improved performance. Requirments of the project can be found here.

© Adam Wojtczak Source Code API Docs oorlu-CLI
Frontend:

Designed with simplicity and code independence in mind, requiring zero dependencies to run. This ensures best possible performance of the app. JS and SCSS packaging was achieved using WebPack module bundler.

Privacy Policy

Cookies

We use Google Analytics to to help monitor the website thus _ga cookie is stored. In order to store your consent of _ga cookie we also use technical cookie hasConsent. You can inspect or delete your cookies by going to Storage Inspector in your browser (Shift + F9 on most browsers).

Creating short link

Creating short link creates record in database that stores long url and datetime when created, we do not gather any additional information about user. When you create a Oorlu Link, the original URLs you have shortened and the corresponding Oorlu Links are publicly available for those that have short links (random 7 char token), example: oor.lu/xYCTiiI.

Do not use Oor.lu for confidential information. We will do our best to protect long links we store but there is no ultimate protection and such things as brute force attacks (someone trying out combinations of url tokens many times) may happen that could expose your long link to someone that you didn't share short url with.

We reserve right to delete links if we see they are leading to any malicious websites and in general we do not guarantee that any link will persist in our database.

You can contact us via social media, see `About` menu if you want to delete link you posted - we will do our best to help you.

Opening short link

Upon opening oorlu short link we store IP, datetime and HTTP referer as part of statistics. These are to monitor network load and performance of website and to identify any spam / malicious use. Currently this info is not available for public users.

Upon clicking we redirect you to long url that was entered when creating link.

IMPORTANT

Under no circumstances we are responsible for contents of long url website author of short link has provided. If you want to see where short url leads prior to opening it you can go to https://oor.lu/{token}/examine where token is 7 char long from short link, example: oor.lu/tXVM8w0/examine
